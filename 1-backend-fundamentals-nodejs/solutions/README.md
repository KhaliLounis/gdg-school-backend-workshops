# Solutions

These are example solutions for the mini-task. There are many correct ways to solve it!

## Files

- `mini-task-solution.js` - Complete solution for the personal backend task
- `advanced-example.js` - Slightly more advanced version with extra features

## Key Learning Points

### Route Handling

- Use `req.url` to check which route was requested
- Handle different cases with if/else statements
- Always provide a fallback for unknown routes

### Response Management

- Set appropriate headers with `res.writeHead()`
- Use `res.end()` to send the final response
- Keep responses clear and user-friendly

### Development Tips

- Use `console.log()` to debug and understand what's happening
- Test all your routes in the browser
- Restart the server after making changes

Remember: The goal is understanding, not perfection!
