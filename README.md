# Backend Workshops - GDG School

A 4-session workshop series covering Node.js and backend fundamentals.

## Sessions Overview

1. **Backend Fundamentals & Node.js** - What is a backend, Node.js basics, first HTTP server
2. **Express Basics** - Framework introduction, routing, middleware
3. **MongoDB & Mongoose** - Database integration, data modeling
4. **CRUD API** - Building a complete REST API

## Prerequisites

- Basic JavaScript knowledge
- Node.js installed (v18+ recommended)
- Code editor (VS Code recommended)
- Terminal/command line familiarity

## Getting Started

Each session folder contains:

- README with learning objectives
- Code examples and demos
- Hands-on exercises
- Solutions (in separate folders)

Navigate to each session folder and follow the README instructions.
