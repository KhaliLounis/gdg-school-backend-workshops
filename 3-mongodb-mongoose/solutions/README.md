# Solutions

This folder contains complete solutions for the mini-task.

## Files

- `task-api-solution.js` - Basic solution with Create & Read operations
- `advanced-task-api.js` - Advanced solution with bonus features

## Running Solutions

```bash
npm run solution           # Run basic solution
npm run solution:dev       # Run with nodemon
```

## Learning Tips

1. Try to complete the mini-task on your own first
2. If stuck, look at specific parts of the solution
3. Compare your approach with the solution
4. Try the bonus challenges after completing the basic task
